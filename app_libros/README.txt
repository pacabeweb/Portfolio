Aplicación de gestión de libros desde localhost

1. Abrir bbdd.php para crear nuestra base de datos.
2. Abrir tabla-libros.php y tabla-usuarios.php para crear las tablas que alojarán la información.
3. Abrir la aplicación: app_libros -> index.php
   Creamos nuevo usuario para poder comenzar a utilizar la app.
   Una vez en la página principal, introducimos nuevo libro y, a partir de ahí, podemos realizar
   acciones de buscar, modificar registros o eliminarlos.

Creada con html, css y php.
